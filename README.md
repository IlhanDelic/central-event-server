
# Central event server

This is the "client" server that communicates with a "central" server.

This server will pick up messages send by [satellite-server](https://github.com/IlhanDelic/satellite-server).


# Installation

The software is installed using:

    npm install express

# Setup

There are no configuration options. Start the server using:

    node ./src/server.js
